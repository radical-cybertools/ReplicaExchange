RepEx: Replica Exchange simulations Package

RepEx is a flexible Python package which is aimed at enabling users to perform Replica Exchange simulations at scale which is not attainable by stand-alone MD applications. RepEx relies on a concept of Pilot Job to run RE simulation on HPC clusters and Grids. Namely, RepEx is using RADICAL-Pilot [1] Pilot Job system for execution of it's workloads. Currently RepEx supports two MD codes as it's application kernels - NAMD [2] and Amber [3]. Each of these kernels can be used to run RE simulation conforming to one of three currently supported RE schemes. For more information please visit About section of this website. RepEx is modular object-oriented code, which is designed to facilitate development of extension modules by it's users.

[1] https://github.com/radical-cybertools/radical.pilot
[2] http://www.ks.uiuc.edu/Research/namd/
